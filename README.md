# portfolio-paramesh

A static portfolio website for Paramesh.

## Usage

Open `index.html` in your web browser to view the portfolio.

## Structure

- `index.html`: Main HTML file
- `styles.css`: CSS styles for dark theme and responsive design
- `assets/`: Folder for images and other assets

## Customization

Edit `index.html` to update content and `styles.css` to change styling.
